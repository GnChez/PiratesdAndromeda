┌─ PASO 1: SPLASH SCREEN (Automático) ──────────────────┐

│ Coloca tu imagen aquí:                                │

└────────────────────────────────────────────────────────┘

┌─ PASO 2: ICONO DE APP (Elige una opción) ─────────────┐
│                                                        │
│ OPCIÓN A: Android Studio (Recomendado)               │
│ →  File → New → Image Asset                           │
│ →  Selecciona relieve.png                             │
│ →  Android Studio genera todo automáticamente         │
│                                                        │
│ OPCIÓN B: Script Automático                           │
│ →  Coloca relieve.png en: app/                        │
│ →  Ejecuta: setup_app_icon_windows.ps1                │
│    (o setup_app_icon_windows.bat)                     │
│                                                        │
│ OPCIÓN C: Online (Sin instalar nada)                 │
│ →  https://romannunik.github.io/AsyncAssetStudio      │
│ →  Sube relieve.png                                   │
│ →  Copia archivos en: app/src/main/res/mipmap-*/     │
│                                                        │
└────────────────────────────────────────────────────────┘

┌─ PASO 3: USAR LA APP ──────────────────────────────────┐
│                                                        │
│ 1. Abre Android Studio                                │
│ 2. Build → Sync Now (si aparece)                      │
│ 3. Ejecuta (Shift + F10)                              │
│                                                        │
│ Verás:                                                │
│ ✨ Tu imagen en la SPLASH SCREEN al iniciar           │
│ 🎨 Tu imagen como ICONO en el menú de apps            │
